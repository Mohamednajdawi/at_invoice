Was Sie bekommen

Einfache REST API für ebInterface XML Generierung

JSON to XML

Senden Sie strukturierte JSON-Daten, erhalten Sie ebInterface 6.1 konformes XML zurück.

Validierung

Automatische Validierung aller Pflichtfelder und Formate (VAT-ID, IBAN, BIC).

Schnell

Response-Zeit unter 200ms. Go-Backend auf Railway gehostet.

Keine Speicherung

Ihre Rechnungsdaten werden nicht gespeichert. Verarbeitung nur im RAM.

API-Key Auth

Einfache Authentifizierung per X-API-KEY Header. Verwaltung über Stripe.

Web Generator

Drag & Drop Interface für manuelle Uploads ohne Code.
